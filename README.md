# AquaGuard - Smart Community Health Monitoring System

## Problem Statement ID: SIH25001

A comprehensive IoT and AI-powered solution for monitoring water-borne diseases in rural and tribal areas of Northeast India.

## Project Structure

- `backend/` - Node.js API server with Express.js
- `frontend/` - React/Next.js web application
- `iot/` - IoT device code and simulators
- `ai-ml/` - Machine learning models for prediction
- `mobile-app/` - React Native community reporting app
- `ar-vr/` - AR/VR educational experiences
- `database/` - Database schemas and migrations
- `docs/` - Project documentation
- `deployments/` - Deployment configurations

## Setup Instructions

### Backend
1. Navigate to `backend/`
2. Run `npm install`
3. Copy `.env.example` to `.env` and configure
4. Run `npm start`

### Frontend
1. Navigate to `frontend/`
2. Run `npm install`
3. Run `npm run dev`

### IoT Simulators
1. Navigate to `iot/simulators/`
2. Install required Python packages: `pip install paho-mqtt`
3. Run `python sensor_simulator.py`
